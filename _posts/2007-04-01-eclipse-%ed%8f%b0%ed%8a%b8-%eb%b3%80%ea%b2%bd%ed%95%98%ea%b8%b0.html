기본 폰트가 지겨우신 분들은 한번 쯤 바꿔보시는 것도 괜찮지 않을까요? 흐흣

전체 폰트를 싹 다 바꾸고 싶으실 땐 Basic -> text font를 바꿔주시면 됩니다. Java 파일의 폰트만 바꾸고 싶으실 때는 Java -> textfont를 바꿔주시면 되지요.

나중에 다시 되돌리고 싶으실 땐 reset버튼을 사용해주시면 됩니다. 그럼 다시 예전의 기본 폰트로 돌아가게 됩니다.

저처럼 1024 * 768 해상도를 사용하는 개발자 분들은 폰트를 바꿈으로 해서 부가적으로 얻을 수 있는 성과가 하나 있습니다.
가로 길이가 짧아서 짤립니다. 폰트를 Corbel로 바꿨더니 가로 길이가 짧아지고 세로 길이도 짧아졌습니다. 흐흐.. 글자채도 볼만하고 만족스럽습니다.

Eclipse IDE for Java Developers를 설치하고 나서

update 사이트를 이용하여 다음의 플러그인들을 설치했습니다.
Web Tools, Subversive, Spring IDE, AnyEditTools

다운로드 하여 설치한 플로그인들
Copy Fully Qualified Class Name, ResourceBundleEditor, SwitchUnit

설치 하기 전에 export로 빼두었던 코드 탬플릿과 포맷을 import합니다.
업데이트 사이트도 이런식으로 export 해두었다가 새로운 이클립스를 설치할 때 업데이트 사이트 목록을 새로 입력할 필요 없이 읽어 들인 뒤 선택하시면 됩니다.

흠.. 이제는 기본 인코딩 설정과 톰캣 경로를 잡아주고 svn 저장소 위치 등록 해주고 소스 코드만 다운 받으면 개발 환경을 3.2 에서 3.3으로 완전히 전환하게 됩니다. :)

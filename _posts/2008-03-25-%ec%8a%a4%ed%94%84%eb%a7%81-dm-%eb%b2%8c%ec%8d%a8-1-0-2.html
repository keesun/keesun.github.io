한달마다 버전이 올라가고 있습니다. 흠 어서 공부해달라고 저를 원하는 스프링 DM의 눈빛이 제 눈에 정말 너무 훤 하네요. 그래서..Changelog라도 살펴봤습니다. 버그 수정해서 올린 버전이라 그런지 몇 개 없네요.

General
* manifest를 내부 페키지들을 노출시키지 않도록 조정했고 스프링과 스프링 OSGi 용 버전을 제공한다.

Package org.springframework.osgi.context
* AbstractOsgiBundleApplicationContext는 인터페이스만 사용해서 application context를 OSGi 서비스로 등록한다.

Package org.springframework.osgi.extender
* 서비스 종속성 대기 로깅 레벨을 DEBUG에서 INFO로 높였다.

Package org.springframework.osgi.util
* BundleDelegatingClassLoader은 번들 업데이트 할 때 클래스 캐슁을 막기 위해서 기본 hasshCode/equals 메소드를 사용한다.

Package org.springframework.osgi.test
* 가능하면 설치된 번들들은 뒷단의 리소스 URL이 설정된 위치를 가진다. 설명을 참조하라(?)

자주 쓰는 기능 중에 단축키가 설정되어 있지 않은 것들이 있을 수 있습니다.
리팩터링 기능 중에도 대 부분이 단축키로 지정되어 있지 않습니다. 하지만 리팩터링 메뉴 자체에 Alt + Shift + T로 단축키가 걸려있기 때문에 이 단축키를 써서 해당하는 리팩터링 메뉴를 실행할 수 있지만 종종 귀찮을 수도 있습니다.
그리고 새로 설치한 플러그인에 있는 기능 중에서도 단축키가 없는 경우가 있습니다. 'Ctrl + Alt + 화살표 위,아래' 를 사용하여 여러줄 복사를 할 수 있지만 그렇게 복사를 하면 바로 아래나 위에 해당한 줄이 복사되어 붙여 넣어지기 때문에 'Alt + 화살표 위, 아래'를 사용하여 다시 원하는 위치로 끌고 가야 합니다. 이 때 유용한 플러그인이 바로 예전에 대엽님께서 찾아 주셨던 Enhancements Editor 입니다. 이 플러그인을 다운로드 하여 설치하시면 다음과 같은 메뉴가 생깁니다.
하지만 안타깝게도.. Copy Line에 단축키가 설정되어 있지 않습니다.;;

Window -> Preferences, 왼쪽 메뉴 중에 General -> Keys 를 클릭하시면 현재 단축키가 등록되어 있는 메뉴들이 나옵니다.
여기서 include unbound command에 체크를 하시면 단축키가 등록되어 있지 않은 메뉴들도 같이 보여집니다. 그리고 나서 tpye fielter text라는 곳에 원하는 기능을 검색합니다. copy line이라고 검색했습니다.
Binding 란에서 해당 메뉴를 실행할 단축키를 입력합니다. 입력하는 방식은 단축키를 누르듯이 합니다. 타이핑 하는 것이 아닙니다. 그러면 When이라는 메뉴가 생기고 text를 편집할 때라고 선택하고 Apply -> OK 를 선택합니다....

하지만... 사용할 수 없습니다;; 왜 이러는건지.. 3.3 버그인가;
keys 에는 분명 등록 되어 있지만;; 메뉴에는 등록되지 않고 실제 사용도 안 됩니다. 버그겠죠??;;

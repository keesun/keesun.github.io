1. 업데이트 사이트를 사용해서 TPTP 플러그인을 설치했습니다.
http://eclipse.org/tptp/updates/site.xml


최신버전으로 보이는 4.4.0.3만 선택하여 설치했습니다.

2. 이제는 Agent Contoller를 설치할 차례입니다.

참조한 글에서 Agent Controller를 다음과 같이 설명하고 있습니다.

에이전트 컨트롤러는 애플리케이션을 이클립스에서 실행하고, 프로파일링 데이터를 추출하기 위해 해당 애플리케이션과 상호 작용하는 TPTP의 핵심 컴포넌트다.

설치하려면 일단 파일을 받아야합니다.
파일은 http://www.eclipse.org/tptp/home/downloads/ 여기서 받을 수 있습니다.

3. Agent Controller를 설치하고 나서, 설정에 들어갑니다.


/가 아니라 \인데 리눅스랑 햇갈렸습니다.
Anyway, 위 명령을 실행하면 다음과 같이 기본 설정에 관한 질문을 던져옵니다. 저는 그냥 Enter만 쳐서 기본설정을 사용하기로 했습니다.


4. 이제 Eclipse에 프로젝트를 생성하고 참조한 문서에 있는 자바 애플리케이션을 붙여넣었습니다.

소스코드에 자잘한 에러들이 있어서 수정이 필요했습니다.
dk030000000001.java
5. 이제 TPTP를 사용해 봐야겠습니다. 그전에 위에서 설치했던 Agent Controller를 먼저 실행합니다.

bin 디렉토리에 있는 ACServer.exe 파일을 실행해 줍니다.

그리고 Eclipse에서 프로프라일러를 사용하여 해당 애플리케이션을 실행합니다.

6. 결과 확인


우와..굉장히 다양한 메뉴를 제공해줍니다. 메모리, 호출 스택, UML, 수행 시간, 등등등 너무 많아서 차근차근 살펴봐야겠습니다.

7. Agent Controller 끄기

콘솔에서 ACServer.exe -shotdown을 입력합니다.

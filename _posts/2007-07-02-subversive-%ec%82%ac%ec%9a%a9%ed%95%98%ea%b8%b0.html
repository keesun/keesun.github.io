Eclipse에서 svn를 편하게 사용할 수 있도록 도와주는 플러그인 입니다. 설치는 다음의 업데이트 사이트를 이용하면 간단하게 할 수 있습니다.
http://www.polarion.org/projects/subversive/download/update-site/

1. 소스 코드 다운받기
show view -> svn -> svn repository 를 선택합니다.
단축키 :: Alt + Shift + Q, Q
2. 새로운 저장소 위치 추가
3. 저장소 URI와 ID, Password 또는 private key를 입력합니다.
4. 다운 받을 소스 코드를 찾아서 check out 합니다.
5. 소스코드 변경을 적용하고 싶다면 commit을 하면 됩니다.
이때 중요한 건 commit 하시기 전에 꼭 update를 해봐야 합니다.
작업 하시기 전에도 꼭 update를 해서 최신 코드를 받은 다음에 작업을 해야 합니다.

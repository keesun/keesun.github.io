_M#]이밖에 유용한 단축키들 :


Ctrl + F11 <- 이전에 Run 했던것 바로 다시 Run 해줌(Alt + Shift + X + T 네 개나 누를 필요없슴)

Ctrl + L <- 줄번호를 입력하는 창이 뜨는데 원하는 줄로 바로 이동 시켜줌

Ctrl + Shift + 상/하 <- 다음 멤버로 바로 이동시켜줌

Ctrl + Alt + 상/하 <- 커서가 위치한 줄을 복사해서 바로 아래에 붙여넣어줌(블럭으로 지정하여 사용가능함)

Ctrl + O <- 클래스 Outline을 팝업창으로 띄워주며 검색도 가능합니다.
Ctrl + Shift + L <- 모든 단축키 보기. 혹시 까먹었거나 예상했던 것이 있는지 찾아 볼 때 좋을 듯

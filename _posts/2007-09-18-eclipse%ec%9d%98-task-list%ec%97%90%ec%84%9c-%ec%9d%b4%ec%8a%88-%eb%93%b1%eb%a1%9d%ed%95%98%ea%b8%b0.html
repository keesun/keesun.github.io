새로운 이슈를 등록하는 폼이 지원되지는 않습니다. 다만 이슈를 등록하는 페이지로 바로 이동을 시켜줄 뿐입니다.

쿼리를 우클릭하고 New -> Task를 선택합니다.


그러면 새로운 Task를 추가할 Task Repository가 나타납니다. 만약에 여기서 Task Repository 뷰에서 추가한 Repository목록이 나오지 않으면, Task Repository를 등록하실 때 New Task URL을 설정하지 않으셨기 때문입니다. Task Repository 뷰로 돌아가셔서 해당 URL을 설정해 주시면 됩니다.

Next -> Finish를 선택하면 브라우저에 구글 코드의 Create Issue 화면이 나타납니다.

이 화면에서 새로운 이슈를 등록하고 Task List 뷰에서 쿼리를 다시 F5로 싱크로나이즈 해주면 새로운 태스크가 추가 된 모습을 확인할 수 있습니다.

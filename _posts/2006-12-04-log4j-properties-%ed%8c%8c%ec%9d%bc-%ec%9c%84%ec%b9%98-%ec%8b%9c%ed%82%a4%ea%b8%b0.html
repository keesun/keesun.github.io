class 파일들이 생성되는 폴더에 두면 됩니다. 저번에 작성했었던 log4j 설정하기에서 소스코드 폴더 src랑 test 두 곳에 모두 있어야 했었던 것을 지우고 bin 폴더에 새로 만들었습니다.

class 파일들이 모여있는 bin 폴더는 Eclipse의 패키지 익스플로러가 아닌 네비게이터에서 보입니다.

log4.properties 파일은 class 패스가 잡혀있는 어느 곳에 두어도 괜찮으며 jar파일 안에 넣어 두어도 되며 웹 어플리케이션의 경우에 web.xml 파일에서 지정한 위치에 두면 된다고 합니다. 이 경우는 영회형 블러그에서 본 것 같네요.

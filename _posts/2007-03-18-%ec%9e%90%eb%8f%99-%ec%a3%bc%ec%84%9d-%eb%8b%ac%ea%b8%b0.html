팀플을 시작하게 되서 평소 전혀 신경쓰지 않았던 주석을 달아야 하는 상황이 발생했습니다. 누가 작성한 코드인지를 알아야 하기 때문에 @author가 필요합니다.

Java -> Code Style -> Code Templates -> Comments -> Types를 클릭하고 Edit 버튼을 선택해서 자신의 이름을 @author 뒤에 적어 줍니다.

그리고 다음 부터 새로운 클래스를 추가할 때 다음의 옵션을 꼭 체크해 줍니다.

또는 매번 체크하기가 귀찮기 때문에 defalut로 새로운 type을 만들 때 마다 주석을 붙여 주도록 설정할 수 있습니다.
Java -> CodeStyle 에서 "새로운 메소드와 클래스를 추가할 때 자동으로 주석 달기"에 체크해주시면 됩니다.
